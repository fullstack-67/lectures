# Typescript

- `npx tsc --init`
- Note the `outDir` and `include` fields in `./tsconfig.json`.
  - Once we use `outDir` option any non-`./src` folder needs to be included, including the root.

---

`./tsconfig.json`

```json
{
  "ts-node": {
    "require": ["tsconfig-paths/register"]
  },
  "compilerOptions": {
    // ...
    "outDir": "./dist",
    "baseUrl": "./",
    "paths": {
      "@src/*": ["./src/*"],
      "@db/*": ["./db/*"]
    }
    // ...
  },
  "include": ["./*", "./src/**/*", "./db/**/*"]
}
```
